Challenge Conversor
Rodrigo Brenes Calderón

Este programa permite hacer conversiones entre diferentes tipos de monedas, también coversiones de temperatura y de distancia.
En realidad es bastante sencillo, simplemente se selecciona el tipo de conversión de que se quiere hacer (Moneda, temperatura, distancia), se escoge el tipo inicial de datos y el tipo de datos al cual queremos convertir, después simplemente ingresamos la cantidad a convertir y presionamos el botón "Convertir" y tendremos el resultado. El programa no permite agregar valores que no sean números o el signo de . si se ingresan más de un punto, el programa saca un mensaje de error.
